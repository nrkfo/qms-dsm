# QMS-DSM - Журнал обновлений (Changelog)

В данном файле фиксируются все ключевые изменения, внедренные в систему управления качеством QMS-DSM.

## [1.2.0] - 2026-04-24 14:27
### Добавлено (New Features)
- **Продвинутый Аудит**: Журнал действий теперь сохраняет состояние записи "ДО" и "ПОСЛЕ" изменений для полной прозрачности.
- **Политика AQL**: Добавлена возможность выбора режима контроля (Normal, Tightened, Reduced) в настройках с индикацией в модуле.
- **Управление данными**: Внедрена система автоматической очистки (Retention), удаляющая логи старше заданного количества дней.
- **Планировщик обслуживания**: Система автоматически выполняет проверку базы данных в заданное время (например, 03:00).
- **Детализация Дашборда**: Реализована возможность просмотра подробного списка записей по каждому модулю прямо с главного экрана.
- **Мастер-данные в замерах**: Модули EPS, Крышки и Комплектующие теперь связаны со справочниками Поставщиков и Артикулов (выбор из списка).
- **Гибкая валидация этикеток**: Длина SN, MN и EAN теперь настраивается через Админ-панель.
- **Полный аудит удалений**: Любое удаление записи теперь фиксируется в журнале с сохранением удаленных данных.
- **Оптимизация БД**: Добавлены индексы (date, lot_id) для ускорения работы дашборда при больших объемах данных.

### Исправлено (Bug Fixes)
- **Синхронизация Дат**: Исправлена проблема, при которой записи пропадали из списка из-за разницы в форматах даты (ISO vs RU). Теперь поиск работает по обоим форматам.
- **Реал-тайм метрики**: Дашборд теперь агрегирует данные напрямую из таблиц логов, исключая задержки и рассинхрон.
- **Безопасность**: Исправлена ошибка в модальном окне пароля при редактировании записей.
- **Стабильность**: Исправлен критический баг (черный экран) в журнале AQL, вызванный обращением к неопределенным настройкам.

### Удалено (Clean up)
- Удалены неиспользуемые настройки (Маска SN, Дневной план KPI, Глобальные настройки отчетов), чтобы упростить интерфейс.

## [1.1.0] - 2026-04-23
### Добавлено
- **Централизованный Админ-панель**: Управление пользователями, лотами, поставщиками и артикулами в одном месте.
- **Импорт Артикулов**: Добавлена функция загрузки номенклатуры из Excel.
- **Синхронизация имен**: Названия модулей в админ-панели, сайдбаре и на дашборде приведены к единому стандарту.
